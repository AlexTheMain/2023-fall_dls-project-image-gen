Скачать необходимые файлы по ссылке https://disk.yandex.ru/d/i3Lbn4P7VLeZNw

1. Сохранить содержимое папки TG_BOT к себе на компьютер
2. В файле Dockerfile во 2-й строке ENV BOT_TOKEN="YOUR_TOKEN" YOUR_TOKEN заменить на действительное значение токена (его можно получить в своем телеграмм-аккаунте у бота BotFather)
3. Открыть терминал из каталога где лежит Dockerfile
4. С помощью терминала собрать образ контейнера командой "docker build -t cyclegan-bot . "
5. Запустить контейнер из образа командой "docker run cyclegan-bot"
6. В Telegramm отправить боту команду '/start'. Если придет ответ "Hello!" значит все работает и можно отсылать в бот изображение для стилизации.
7. В ответ бот пришлет 2 стилизованных изображения с указанием типов генераторов с помощью которых онибыли созданы.
